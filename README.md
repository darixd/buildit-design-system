# BuildIT Design System

Shared CSS tokens, typography and components for BuildIT applications (GESTUP, Presenze).

## Usage

Add as a git submodule:

```bash
git submodule add git@github.com:darixd/buildit-design-system.git shared
```

Then import in your CSS:

```css
@import "../shared/buildit-shared.css";
```

## Contents

- **buildit-shared.css** — CSS custom properties (colors, typography), shadcn/ui HSL tokens, and shared component styles
